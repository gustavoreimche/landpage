Dinner Delivery

O Dinner Delivery é uma plataforma de e-commerce de última geração com um sistema completo que simplifica a experiência de compra online, oferecendo uma variedade de recursos incríveis.

Desenvolvido utilizando tecnologias de ponta, o DinnerDelivery foi construído com Angular e Node.js, combinando o poder do frontend e do backend para oferecer uma experiência perfeita aos usuários. Essa solução robusta permite que os clientes realizem pedidos de forma fácil e rápida, além de desfrutarem de um processo de entrega eficiente.


Barber Manager

O Barber Manager é um gerenciador de barbearias completo e sofisticado. Com ele é possível criar agendamentos, cadastrar clientes, gerenciar pagamentos e despesas, exportar relatórios e muito mais!
